## 2026-06-23 — State Estimators + Trajectory Generator
**Last commit:** `bc4e09a` — "docs: enhance controller/plant/estimator/trajectory base classes with detailed descriptions and usage examples"

**Files added:**
- `kalman_filter.py` — `KalmanFilter(StateEstimator)` with full predict-update cycle
- `luenberger_observer.py` — `LuenbergerObserver(StateEstimator)` with observer gain

**Files evolved:**
- `components.py` — Added `TrajectoryGenerator` ABC, enhanced docstrings on all ABCs

**Decisions:**
- **KalmanFilter.** Full discrete-time linear KF with predict (A, B, Q) and update (C, D, R) steps. Tracks posterior state estimate `x_hat` and error covariance `P`. Defaults C to identity, D to zeros.
- **LuenbergerObserver.** Deterministic observer with configurable gain L. Implements `x̂ₖ₊₁ = A x̂ₖ + B uₖ + L (yₖ − C x̂ₖ − D uₖ)`.
- **TrajectoryGenerator ABC.** Abstract interface for generating trajectories from start to end over a duration. No concrete implementation yet.
- **Enhanced ABC docstrings.** All four ABCs (Controller, Plant, StateEstimator, TrajectoryGenerator) now have detailed docstrings with usage examples.

**Working:** Kalman filter (predict-update cycle). Luenberger observer. All four ABCs with documentation.
**Not working:** No concrete TrajectoryGenerator implementation. No MuJoCo Python integration. No combined base+arm control.
